# Installation

1. Install Docker
2. You only need docker-compose.yml file. Download docker-compose.yml
3. Code to start containers: <br>
   `$ docker-compose up -d`
4. Now you can use the program at http://localhost:5000/
5. To kill the containers, <br>
   `$ docker-compose down`
